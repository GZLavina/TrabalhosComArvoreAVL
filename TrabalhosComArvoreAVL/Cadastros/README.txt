Desenvolvido como trabalho avaliativo para a disciplina de
Algoritmos e Programação: Árvores e Ordenação

 - Esse trabalho foi construído em cima da árvore AVL produzida
anteriormente.

 - Objetivos:
 * Desenvolver um programa que leia cadastros de arquivos csv
e insira-os em árvores AVL.
 * Todos os registros devem ser mantidos em uma estrutura simples (um array),
e a árvore deverá conter índices para acessar rapidamente esse registro.
 * Deve haver uma árvore para guardar o CPF, uma para guardar o nome, e uma
para guardar a data de nascimento.
 * A consulta por nome deve mostrar todos os nomes que iniciam com a string
digitada: "j" mostra todos os nomes que começam com "j".
 * A consulta por data deve ser feita com uma data inicial e uma final,
e o resultado será todas as pessoas que nascem entre os dois dias inseridos.
 * Programar a interação com o usuário.

 - Inclui arquivos .csv para testes.
