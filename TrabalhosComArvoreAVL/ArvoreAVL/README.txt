Desenvolvido como trabalho avaliativo para a disciplina de
Algoritmos e Programação: Árvores e Ordenação

 - Objetivos:
 * Desenvolver uma árvore AVL para guardar números inteiros.
 * Programar a interação com o usuário para permitir inserção,
remoção e busca de valores.
 * Mostrar a árvore após cada operação realizada.
 *  A busca deve mostrar também o caminho percorrido para chegar 
no valor desejado.
 * Também mostrar o percurso "pré-ordem", "em ordem" e "pós-ordem".
